Adaptive and Responsive Web Design

Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

Responsive Web Design adalah sebuah teknik yang digunakan untuk membuat layout website menyesuaikan diri dengan tampilan device pengunjung, baik ukuran maupun orientasinya. Jadi tampilan yang berada di desktop komputer dengan tampilan yang diakses melalu SmartPhone misalnya, itu akan berbeda tampilannya. Responsive Web Design Untuk mengetes tampilan yang menggunakan teknik Responsive tersebut tidak harus mengaksesnya melalui mobile device ataupun device lainnya, namun cukup dengan meminimize browser saja ke dalam ukuran mobile device yang diinginkan, kita sudah dapat melihat tampilan website kita pada layar mobile device.

Sedangkan, adaptive web design berarti kemampuan suatu website dalam hal ini tampilan website untuk beradaptasi dengan ukuran layar. Lalu, apa bedanya dengan responsive web design ? Bukankah responsive juga bisa beradaptasi dengan ukuran laya ? Keduanya memang bisa menyesuaikan dengan ukuran layar, namun, dalam hal ini adaptive lebih luas cakupannya, bukan hanya bisa menyesuaikan lebar website dan elemen-elemennya, namun tampilan website bisa menjadi berbeda sama sekali antara tampilan di komputer dan di prangkat mobile.


Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

Iya tentu dengan makin banyaknya pengguna smartphone dan tablet, para website developer harus memikirkan mengenai desain website untuk setiap platformnya apakah akan dibuat sama (RWB) atau dibuat secara custom (AWD). Saya pikir di masa depan tampilan web akan lebih inovatif, mungkin akan banyak animasi yang ditampilkan, video berkualitas tinggi (5k) dll.


Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

Iya saya pikir perlu karena mereka juga memerlukan teknologi untuk membantu kekurangan mereka dalam aktivitas sehari-hari. Untuk membantu mereka mengakses website jelas developer perlu memberikan tools (voice recognition, berbagai sensor, dll) dan juga memberikan kustomisasi pada website mereka untuk orang-orang yang kurang beruntung seperti mereka.
